# PageRoast $497 Deliverable Specification

## What The Customer Gets

### 1. The Roast (Audit)
Detailed breakdown of everything wrong with their current page:
- Above-the-fold analysis
- Headline teardown
- Social proof audit
- CTA analysis
- Objection handling gaps
- Mobile issues
- Technical problems

**Format:** Written document or Loom video walkthrough

### 2. The Rewrite (The Actual Product)
Complete new landing page ready to deploy:
- New headline + subheadline
- Rewritten body copy throughout
- Restructured sections for conversion flow
- New CTA copy at each touchpoint
- Social proof placement recommendations
- Objection-handling sections

**Format:** Clean HTML file they can deploy OR copy doc they can hand to their dev

### 3. The Explainer
Brief notes on WHY each major change was made:
- "Changed headline from X to Y because..."
- "Moved pricing section because..."
- "Added this objection-handler because..."

This helps them understand the thinking so they can apply it to future pages.

## Quality Bar

Before sending, the rewrite must pass these tests:

1. **The "Holy Shit" Test:** Would the customer immediately think "this is so much better"?
2. **The Specificity Test:** Is every headline, CTA, and section specific (not generic)?
3. **The Proof Test:** Is there social proof visible in the first scroll?
4. **The Action Test:** Is the CTA clear, compelling, and repeated appropriately?
5. **The Objection Test:** Are the top 3 customer objections addressed on the page?

If any test fails → fix it before sending.
